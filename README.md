# 测试ML.Net调用Yolov5的Onnx模型
通过yolov5训练自定义数据，然后把模型转成onnx格式，通过ML.Net调用该模型
**使用方法：**
1. 选择模型
2. 选择保存结果路径
3. 标签类别，例如：apple,orange
4. 选择单图识别还是失败图片路径
